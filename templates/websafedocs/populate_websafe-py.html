populate_websafe.py

This file updates the WebSAFE database based from the csv files located in the csv_files folder.

Description Function / Variable
This exports the PostgreSQL database to noah_db.sql file. export_database()
This function commits the changes to the external repository. commit_changes_to_repo()
This function drops the rows in the websafe tables of the database. empty_websafe_db()
This function populates the conveniences WebSAFE table from the csv_files/websafe_conveniences.csv. populate_conveniences()
This function populates the economics WebSAFE table from the csv_files/websafe_economic.csv. populate_economics()
This function populates the demographics WebSAFE table from the csv_files/websafe_demog.csv. populate_demog()
This function populates the education WebSAFE table from the csv_files/websafe_educ.csv. populate_educ()
This function populates the energy WebSAFE table from the 'csv_files/websafe_energy.csv'. populate_energy()
This function populates the land tenure WebSAFE table from the csv_files/websafe_landtenure.csv. populate_landtenure()
This function populates the rh WebSAFE table from the csv_files/websafe_rh.csv. populate_rh()
This function populates the WASH WebSAFE table from the csv_files/websafe_wash.csv. populate_wash()
